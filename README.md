# 📦 Courier Management System

## 📌 Project Overview
The **Courier Management System** is a desktop-based Java application built using **Swing GUI**.  
It allows administrators to manage courier records and users to track shipments easily using a Tracking ID.

The system is designed with a simple role-based interface:
- **Admin**: Can add couriers and update delivery status
- **User**: Can only track courier status

---

## 🎯 Objectives
- Simplify courier management operations
- Provide real-time tracking functionality
- Separate admin and user access levels
- Build a lightweight desktop application using Java Swing

---

## 🛠️ Technologies Used
- **Programming Language:** Java  
- **GUI Framework:** Java Swing (javax.swing)  
- **Data Structure:** HashMap (in-memory storage)  
- **IDE (Recommended):** IntelliJ IDEA / Eclipse / NetBeans  

---

## 🚀 Features
### 🔐 Admin Features
- Secure login (ID & Password)
- Add new courier details
- Update courier status
- View courier information

### 👤 User Features
- Track courier using Tracking ID
- View sender, receiver, and delivery status
